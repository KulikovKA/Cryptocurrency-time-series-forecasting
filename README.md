# Cryptocurrency-time-series-forecasting
Прогнозирование цен криптовалют с использованием Deep Learning. Сравнение производительности LSTM, GRU и BiLSTM на волатильных и структурированных данных.

## Навигация по репозиторию

В данном репозитории находятся следующие материалы:

1.  **`cripta.ipynb`** — Основной Jupyter Notebook. Содержит полный цикл разработки: загрузка данных, предобработка (Scaling, Sliding Window), обучение нейросетей и код для генерации графиков.
2.  **`images/`** — Папка с сохраненными графиками и визуализациями прогнозов, полученными в ходе работы.
3.  **`README.md`** — Файл с описанием проекта.
    > *Здесь собраны **все ключевые метрики и финальные выводы** исследования, чтобы вы могли оценить результаты без необходимости просматривать громздкий файл "cripta.ipynb".*
